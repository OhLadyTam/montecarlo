More about the application

This is a web application. This application is based on Luminus architecture. The application starts on http://localhost:3000/ once it's run. You can run it with Leiningen by typing "lein run" in project root file. The code regarding the simulation is located in the file "montecarlo/montecarlo_simulation.clj". Data is taken from YahooFinance datasource, via an API written in Java. Do a rebuild of the project before the first start of the application.
